# Movie Finder

Proyecto de prueba técnica para Nextep, basado en un buscador de películas interactivo mediante la api de ImDB.

## Ejecución en local

- Clonar el [repositorio](https://github.com/Seeker4264/movie-explorer-app) público de Github.

- Instalar las dependencias mediante [pnpm](https://pnpm.io/) o cualquier otro manejador de paquetes de Node.js (durante el desarrollo se utilizó pnpm, por lo que es recomendable usar éste).

```bash
pnpm install
```

- Ejecutar el proyecto localmente.

```bash
pnpm run dev
```

```bash
pnpm dev
```

- El proyecto debería de ejecutarse en el puerto 5173 por defecto.
